‚ö° Quick Start Guide

  1. Ensure you have a valid kubeconfig file (typically at ~/.kube/config)
  2. Click the {{ACTIVITY_BAR_ICON}} icon in the VS Code activity bar (left sidebar)
  3. Select any cluster to open an interactive dashboard with real-time statistics
  4. Expand clusters to explore namespaces and Kubernetes resources
  5. Click on any resource to view detailed information

üìÅ Organize Your Clusters

Use the Cluster Organizer to customize how your clusters appear in the tree view:

  • Create folders to group related clusters together
  • Set aliases to display friendly names instead of long context names
  • Hide clusters to reduce clutter in your tree view
How to access: Open the Command Palette (Cmd+Shift+P or Ctrl+Shift+P) and search for "Kube9: Cluster Organizer"
